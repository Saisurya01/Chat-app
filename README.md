# Chat App

A simple real-time chat application built using **HTML**, **CSS**, and **JavaScript**, with **Firebase Realtime Database** for message storage and synchronization. This app allows users to send and receive messages instantly through a clean and minimal interface.

**Live Demo:** https://saisurya01.github.io/Chat-app/

---

## Overview

The Chat App is a lightweight web-based messaging application that demonstrates real-time communication using Firebase. Messages sent by one user are instantly reflected for all connected users without refreshing the page.

This project focuses on learning real-time databases, JavaScript event handling, and dynamic UI updates.

---

## Features

- Real-time messaging using Firebase
- Messages update instantly without page refresh
- Simple and clean chat interface
- Responsive design
- No backend server required

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Firebase Realtime Database
